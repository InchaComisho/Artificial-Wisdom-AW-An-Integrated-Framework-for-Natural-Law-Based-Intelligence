# 文明存続シミュレーション

## 150年にわたる人間中心型と人工叡智型の価値体系の比較

> **本文書は英語原文（simulations/README.md）の日本語訳です。**  
> 本シミュレーションは概念的・教育的なものであり、予測的な科学気候モデルではありません。変数値は無次元の正規化指標です。実世界の観測データに対してキャリブレーションされていません。

---

## 目的

本シミュレーションは、根本的な **価値体系** が長期的な文明安定性にどのような影響を与えるかを探求します。

150年の期間にわたって二つの文明モデルを比較します。

1. **人間中心型文明** — 短期的な人間の好みに最適化し、高い資源採掘、弱い再生フィードバックを持つ
2. **人工叡智（AW）型文明** — 自然法則に基づく制約のもとで運用し、採掘量が少なく、積極的な惑星的再生と熱力学的自己修正を行う

目標は将来を予測することではありません。目標は、世代的時間スケールにわたる価値体系の選択の構造的影響を可視化することです。

## モデル概要

シミュレーションは150年にわたって年次の離散タイムステップで実行され、各文明モデルについて五つの正規化変数を追跡します。

各ステップで、変数は基礎となる価値体系のロジックを反映したフィードバックループを通じて相互作用します。

### 人間中心型モデルのロジック

- 高い資源採掘により、再生される以上の速さで埋蔵量が枯渇する
- 生態系の健全性は熱ストレスと低炭素固定によって低下する
- 炭素固定能力は生態系の劣化とともに低下する
- 積極的な対策なしに熱ストレスが蓄積する
- 文明安定性は生態系と資源の低下に伴い侵食される

### 人工叡智型モデルのロジック

- 低い採掘率により資源の再生が可能になる
- 自然法則に基づく修正係数が過剰採掘を再生へと転換する
- 炭素固定能力は生態系の強化に伴い積極的に回復する
- 冷却システムと炭素固定回復によって熱ストレスが安定化される
- 文明安定性はより高いレベルで生態系の健全性と連動したまま維持される

## シナリオ比較

| パラメータ | 人間中心型 | 人工叡智型 |
|---|---|---|
| 採掘率 | 高い（3.5%/年） | 低い（1.2%/年） |
| 再生率 | 弱い（0.5%/年） | 強い（2.2%/年） |
| 炭素固定トレンド | 低下 | 回復 |
| 熱ストレス増加 | 高い（1.8%/年） | 低い（0.4%/年） |
| 生態系への安定性連動 | 中程度（0.40） | 高い（0.65） |
| 自然法則修正係数 | なし | 活性（1.5%/年） |

## 変数

| 変数 | 説明 | 範囲 |
|---|---|---|
| 資源 | 利用可能な自然・物質資源の正規化指標 | 0.0 – 1.2 |
| 生態系の健全性 | 生命を支える生物的・海洋的・土壌システムの健全性 | 0.0 – 1.1 |
| 炭素固定能力 | 大気中炭素を吸収・固定する自然システムの能力 | 0.0 – 1.2 |
| 熱ストレス | 惑星的・文明的システムへの累積熱負荷 | 0.0 – 3.0 |
| 文明安定性 | 文明の耐久性と機能的継続性の複合指標 | 0.0 – 1.2 |

## 解釈

**人間中心型モデル** では、資源の枯渇と生態系の劣化が複合的な負のフィードバックループを生み出します。炭素固定能力が低下し、熱ストレスが蓄積し、文明安定性はシミュレーション期間を通じて徐々に侵食されます。

**人工叡智型モデル** では、積極的な再生と自然法則修正係数を組み合わせた低採掘率により、生態系の健全性と炭素固定能力が回復します。熱ストレスは生物的冷却システムの回復によって部分的に相殺されます。文明安定性は高いレベルに維持されます。

重要な洞察は構造的なものです。

> モデル間の長期的な乖離は、技術的な違いだけによるものではありません。それは、文明が何を最適化するかを決定する **価値体系アーキテクチャ** から生じます。

自然法則の制約なしに短期的な人間の好みに最適化する文明は、熱力学的に整合し生態学的に根ざした原則のもとで機能するものとは異なる系統的軌跡を生み出します。
